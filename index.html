CSS-Snippet Cheatsheet

Ever have trouble recalling the exact syntax for your favorite CSS code? Give it a permanent home and add it to this page! Select any snippet bellow and it will automatically select all of the code for you to copy.

Flex Row

Use these three properties to create a Flexbox row layout

      
        .row {
            display:flex;
            flex-direction: row;
            flex-wrap:wrap;
         }

Flexboc Column

Use this to create a Flexbox column layout.

.row { display:flex; flex-direction: row; flex-wrap:wrap; }

Css Grid Layout

Build a 12-column layout using CSS Grid.

.grid { display: grid; width: 100%; grid-template-columns: repeat(12,1fr); }

Liner gradient

This will create a background linear gradient from top to bottom.

.linear-gradient-background{ background-image: linear- gradient( rgba(232, 102, 236, 0.3)0%, rgba(232, 102, 236, 0.6)100% ); }

Box Transition

Use transition and box shadows to glow.

.code-card .card-header { border-radius:8; transition: all 0.5s ease-in- out; } .code-card:hover .code-card:hover .card-header{ box-shadow: inset 0px 0px 8px rgba(232, 102, 236, 1), 0 0 15px rgba(232, 102, 236,1) ; }

Overlay Card with Title

Use position properties and negative margins to raise elements higher than their natural starting point.

.card-header { position:relative; margin-top: -20px; }
